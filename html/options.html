Instructions:
  1. Set a Master Password and make it long and diverse (use letters, numbers, and symbols), but remember it! (e.g. h4!!o@nd|23w)
  2. Optional, set a Salt Phrase to increase the security of your generated passwords.
  3. Set the Password Length of the generated domain-specific passwords.
  4. Set a Security Reset Interval which will automatically clear the master password (and salt phrase) every so often (ideal for shared computers).
  5. Save, and click on the icon in the toolbar to get your unique domain-specific password!
Cygen


(optional)


All of the above fields are encrypted in AES-256 (US Government-approved "Top Secret" encryption grade) as soon as you hit "Save"! ;)

Note: if you update any of the above settings (except the Reset Interval), you will need to change your password on all the sites you use passwords generated by Cygen.
